** Python Log Analysis Tool

** Overview
This project is a Python-based log analysis tool designed to detect suspicious authentication activity and identify potential security risks.

** Objective
The goal of this project is to analyze authentication logs to identify abnormal patterns such as repeated login failures, unusual access behavior, and potential unauthorized access attempts.

** Tools Used
- Python

** Features
- Detection of repeated failed login attempts
- Identification of unusual login patterns
- Basic analysis of authentication activity

** Methodology
Log data is parsed using Python to extract key fields such as timestamps, usernames, and login outcomes. The script analyzes patterns to identify anomalies and flag potentially suspicious activity.

** Outcome
This project demonstrates the ability to work with log data, perform basic security analysis, and identify potential threats using Python.
